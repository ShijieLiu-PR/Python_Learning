1. 核心数据类型
    A. del语句
        语法：del 变量名1， 变量名2
        作用：用于删除变量，同时解除与对象的关联关系，如果可能，则释放对象。
        自动化内存管理的引用计数：每个对象记录被变量绑定的次数，当为0时对象被销毁。
    B. 空值对象 None
        表示不存在的特殊对象。
        例如：
            a = None
        用于占位，变量解除绑定。
        name01 = "八戒"
        name02 = "悟空"
        name02 = name01
        name02 = None
        print("name01 is " + str(name01))
        print("name02 is " + str(name02))
    C. 整型int
        十进制：5
        二进制：0b开头，如0b11
        八进制：0o开头，如0o10
        十六进制：0x开头0x1a
        小整数对象池：CPython中，整数-5至256，永远存在小整数

    D. 浮点型float
        小数，包含正数，负数，0.0
        字面值：
            小数：1.0  2.5
            科学计数法：e/E (正负号) 指数 1.23e-2等同于0.0123
    E. 字符串型string
        用了记录文本信息（文字信息）
        字面值：双引号
    F. 复数complex
        由实部和虚部组成的数字
        虚部是以j或者J结尾
        字面值：1j   1+1j   1-1j
    G. 布尔bool
        True, False
    H. 数据类型的转换
        int(),float(),bool() -- 结果为False：bool(0), bool(0.0), bool(None)
        类型自动升级：
        1 + 2.14 返回的结果是3.14， 整型和浮点型求和，结果为浮点型
        1 + 3.0 返回的结果是4.0

        y -= x 等同于 y = y - x
        ...
    C.比较运算符
        <
        >
        ==
        !=

3. 语句
    A.行
        物理行：程序员编写代码的行
        逻辑行：python解释器需要执行的指令
        建议一个逻辑行在一个物理行上。
        如果一个物理行中使用多个逻辑行，需要使用分号分开
        如果逻辑行过长，可以使用隐式换行或者显式换行
        --隐式换行：所有括号的内容换行，称为隐式换行
            括号包括：() [] {}
        --显示换行：通过反斜杠\换行，
    B.选择语句
        作用：让程序根据条件选择性地执行语句。
        语法：
            if condition1:
                block1
            elif condition2:
                block2
            else:
                block3
        说明：elif子句可以有0个或者多个，else语句可以有0个或者1个，且只能放在if语句的最后。
