    封装：
        1.封装数据：将多个基本数据类型复合为一个自定义类型
            优势：复合人类的思考方式
        2.封装功能：对外提供必要的功能，隐藏实现细节
            优势：模块化的编程思想
        3.分而治之，封装变化，高内聚，低耦合
            分解   变化点   类的单一职责  类于类之间关系松散

2. 继承
    语法
        --代码
            class 子类（父类）：
                def __init__(self, 参数列表):
                    super().__init__(参数列表)
                    self.自身实例变量 = 参数
        --说明
            子类拥有父类的所有变量
            子类如果没有构造函数，将自动执行父类的，但如果有，此时必须通过super（）函数调用父类的构造函数，以确保父类的属性被正常创建。
